# node-exemplo

Aplicação Node de exemplo.

## Usabilidade

Para executar a aplicação:

- Instale e configure o [Npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm?ref=sfeir.dev)
- Execute os comandos

```bash
npm install
npm start
```

- Acesse a url <http://localhost:3000> no navegador.
